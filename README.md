## ShannonCode

```
При запуске программы, введите exp.txt и выберите режим "1" для кодирования данного файла, после в файле encoded.txt будет
находиться закодированный текст из exp.txt, который будет весит меньше чем исходный exp.txt
Также после кодировки текстовика можно повторно запустить программу, указав encoded.txt и режим "2", результатом будет
файл decoded.txt, содержащий текст из изначального файла exp.txt
```
